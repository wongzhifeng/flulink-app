# FluLink 项目记忆库

## 🎯 项目核心信息

**项目名称**: FluLink - 异步社交APP  
**当前版本**: MVP v0.2.0  
**开发状态**: MVP开发完成，可部署到生产环境  
**技术栈**: Next.js 15.5.4 + TypeScript + Tailwind CSS + Radix UI  
**部署平台**: Zeabur (Docker容器化)  

## 🏗️ 项目架构

### 核心概念
- **毒株(Strain)**: 内容传播的基本单位，支持4种类型（生活、观点、兴趣、超级毒株）
- **传播系统**: 基于地理位置的异步传播模拟
- **生命周期管理**: 毒株从创建到完成的完整生命周期
- **变异实验室**: 第二阶段功能，支持毒株特性变异

### 目录结构
```
src/
├── app/                    # Next.js App Router
│   ├── page.tsx           # 主页
│   ├── dashboard/         # 主应用界面
│   ├── login/             # 用户登录
│   └── features/          # 功能展示
├── components/            # 组件库
│   ├── auth/              # 认证组件
│   ├── virus/             # 病毒传播核心组件
│   ├── ui/                # UI组件
│   └── layout/            # 布局组件
├── context/               # React Context
├── lib/                   # 工具库
└── types/                 # TypeScript类型定义
```
