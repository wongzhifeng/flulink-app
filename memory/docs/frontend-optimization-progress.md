### 🔧 前端技术架构

#### 技术栈
```
前端框架: React 18 + TypeScript
UI组件: Ant Design 5.x
路由管理: React Router 6
状态管理: Context API + useReducer
HTTP客户端: Axios
样式方案: CSS Modules + Ant Design
构建工具: Vite
```

