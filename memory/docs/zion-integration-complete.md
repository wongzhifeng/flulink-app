## 🎯 **API接口总览**

### **文件上传接口**
```
POST /api/upload/image
- 功能: 上传图片到Zion对象存储
- 参数: image (文件), type (类型)
- 返回: { success, url, fileId, fileName }

POST /api/user/avatar
- 功能: 上传用户头像
- 参数: avatar (文件), userId (用户ID)
- 返回: { success, avatarUrl, fileId }
```

### **地图服务接口**
```
GET /api/amap/geocode
- 功能: 地址转坐标
- 参数: address (地址)
- 返回: 高德地图API响应

GET /api/amap/reverse-geocode
- 功能: 坐标转地址
- 参数: latitude, longitude
- 返回: 地址信息

GET /api/amap/search
- 功能: 地点搜索
- 参数: keyword, city
- 返回: 搜索结果

GET /api/amap/distance
- 功能: 距离计算
- 参数: origins, destinations
- 返回: 距离信息
```
