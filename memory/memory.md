# FluLink 流感发牌手服务 项目记忆库

## 项目概览

**项目名称**: FluLink 流感发牌手服务  
**项目性质**: 异步社交平台，基于AI仲裁者的内容传播游戏化系统  
**核心创新**: 将传统社交的"发布-推送-接收"模式转变为"亮牌-发牌手分析-智能传播"的博弈化交互模式  
**目标用户**: 追求自然社交、平等传播的用户群体  
**愿景**: 让内容像流感一样自然而然地传播，重建人与人之间的真实连接

## 核心产品概念

### 发牌手模式
- **AI仲裁者**: 发牌手不是简单的自动化回复机器人，而是拥有独立思考能力的智慧仲裁者
- **世界规则不变性**: 发牌手遵循的核心世界规则永不可修改，确保公平性
- **动态规则开关**: 管理员可通过后台开关控制细节规则的激活/关闭
- **人格化体验**: 冷静、理性、略带科学家气质的发牌手人格设计

### 亮牌机制
用户通过发送位置和文本内容"亮牌"，发牌手根据既定规则进行分析判断，决定传播策略和回复内容。

### 传播哲学
- **遵循自然**: 传播遵循地理逻辑和自然社会关系
- **平等公正**: 所有用户获得平等传播机会
- **透明可控**: 用户理解规则，管理员可调控参数
- **互利共生**: 付费用户获得效率，免费用户获得公平

## MVP核心功能架构

### 用户端核心功能
1. **位置验证**: 用户首次使用需要发送精确位置（建议街道级精度以上）
2. **亮牌流程**: 输入文本→点击亮牌→等待发牌手分析→接收结果
3. **会话保持**: 发牌手长期记忆用户的传播历史和偏好
4. **成就系统**: 记录用户的传播里程碑和贡献值

### 发牌手服务核心规则（不可修改）

#### 世界法则
1. **公平至上**: 无论付费免费，每张牌都平等对待
2. **自然传播**: 传播路径必须符合地理逻辑和社交常识
3. **真实反映**: 毒性分析基于内容质量，不人为操纵
4. **透明操作**: 传播规则向用户公开，拒绝黑箱操作
5. **道德边界**: 拒绝传播有害、虚假、恶意内容

#### 地理传播协议
```
L1 (本小区)：    即时传播，0延迟
L2 (临近小区)：  5-15分钟延迟，需要L1感染≥20人
L3 (所属街道)：  30-60分钟延迟，需要L2≥2个小区≥15人
L4 (城市区域)：  2-4小时延迟，需要L3≥3个小区传播
跨国传播：       特殊毒株解锁，24-48小时延迟
```

#### 毒性评分算法（固定权重）
```
基础评分 = 内容质量分(40%) + 位置精度分(20%) + 传播历史加权(20%) + 用户信用分(20%)

毒性等级分类：
≤3：仅限本社区传播（低毒性）
4-6：普通传播路径（中毒性）
≥7：超级传播模式解锁（高毒性）
```

### 管理员后台功能
- **规则开关面板**: 直观的开关控制各个规则模块
- **实时数据监控**: 传播数据、用户行为、系统健康度
- **A/B测试管理**: 不同用户群体应用不同规则策略
- **应急响应**: 异常情况紧急关闭或调整规则

## 项目核心理念

**"让内容像流感一样自然而然地传播，让社交回归本质，让连接更有意义。"**

FluLink不是要替代微信，而是要创造一个全新的社交范式：
- 从即时通讯到异步社交
- 从粉丝经济到价值传播  
- 从社交展示到真实连接
- 从算法推荐到自然流动

发牌手的存在让每一次内容传播都充满未知的乐趣，而世界规则的不变性确保了这场游戏永远公平。在这个快节奏、高压力的时代，FluLink为人们提供了一个回归本质的社交空间——在这里，重要的是真诚的分享，而非立即回复；重要的是真实的价值，而非瞬间爆红。

**正如我们的Slogan所说："Spread like flu, link every corner you care"——如流感般扩散，连接你在意的每个角落。**
