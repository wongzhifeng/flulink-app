## 二、发牌手规则引擎 1.0

### 2.1 会话启动规则
```
用户触发条件：
1. 发送地理位置（微信位置消息或手动输入）
2. 发送毒株内容（文本/图片/表单）
3. 系统验证位置精度：
   - 精确到小区（≤100米）：立即启动
   - 街道级别（100-500米）：要求确认
   - 城市级别（＞500米）：拒绝服务

发牌手响应：
1. 2-5秒智能延迟（模拟病毒分析）
2. 显示"病毒培养中..."动态提示
3. 返回区域感染热力图概览
```

### 2.2 毒性评分算法
```javascript
// 毒性分数 = 基础分 + 内容权重 + 地理位置加成
function calculateToxicityScore(content, location, userLevel) {
  let baseScore = 0;
  
  // 文本内容分析（40%权重）
  if (content.text) {
    baseScore += analyzeTextToxicity(content.text) * 0.4;
  }
  
  // 图片传播系数（30%权重）
  if (content.images && content.images.length > 0) {
    baseScore += calculateImageSpreadFactor(content.images) * 0.3;
  }
  
  // 表单结构化度（20%权重）
  if (content.formData) {
    baseScore += evaluateFormStructure(content.formData) * 0.2;
  }
  
  // 用户等级加成（10%权重）
  baseScore += getUserLevelBonus(userLevel) * 0.1;
  
  // 地理位置精度修正
  baseScore *= getLocationPrecisionFactor(location.precision);
  
  return Math.min(10, Math.max(1, baseScore));
}
```

### 2.3 传播路径预测规则
```
一级传播（本小区）：
- 延迟：0分钟
- 触发条件：毒性分数 ≥ 3
- 预计感染：毒性分数 × 2-5人

二级传播（临近小区）：
- 延迟：5-15分钟
- 触发条件：本小区感染 ≥ 20人 或 感染率 ≥ 30%
- 解锁条件：毒性分数 ≥ 5

三级传播（所属街道）：
- 延迟：30-60分钟  
- 触发条件：≥2个临近小区感染 ≥ 15人
- 解锁条件：毒性分数 ≥ 7

超级传播模式：
- 触发条件：毒性分数 ≥ 7.5
- 效果：跨层级传播 + 传播速度 × 200%
- 限制：每日最多触发3次
```

### 2.4 用户分层权益
```
免费用户（区域深耕者）：
- 本小区传播零延迟
- 每日5点精力值（每次亮牌消耗1点）
- 可参与变异计划（收益分成70%）
- 区域投票权（基于历史贡献加权）

付费用户（超级传播者）：
- 基础版（19.9元/月）：
  * 街道层级传播解锁
  * 每日精力值10点
  * 每月3张区域加速券
  
- 高级版（69.9元/月）：
  * 跨国传播权限
  * 自定义毒株演化方向
  * 传播数据分析面板
```
