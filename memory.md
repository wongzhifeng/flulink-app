## 部署规则
### 每次推送代码到Gitee前必须检查
1. **编译检查**
   - `npm run build` 必须成功
   - 无TypeScript编译错误
   - 无重复import语句

2. **组件引用检查**
   - 所有`@/components`导入必须存在
   - 路径别名配置正确
   - 无引用不存在的组件

3. **类型检查**
   - 所有`@/types`导入必须存在
   - VirusStrain等类型正确导出
   - 无类型定义错误

4. **关键文件验证**
   - `src/app/page.tsx` 必须有默认导出
   - `src/app/login/page.tsx` 登录功能正常
   - `src/app/dashboard/page.tsx` 仪表板功能正常

5. **React规范检查**
   - 所有React组件都有默认导出
   - useState等Hook使用正确
   - 无重复的Hook导入
