## 📱 项目概述

FluLink是一个基于地理位置的社交传播应用，让内容像流感一样自然传播。微信小程序版本提供了完整的移动端体验。

## 🔗 API接口

### 后端服务
- **服务器**: `https://flulink-app.zeabur.app`
- **WebSocket**: `wss://flulink-app.zeabur.app/ws`
- **微信验证**: `/wechat/verify`

### 主要接口
- `GET /api/dashboard/stats` - 获取统计数据
- `GET /api/strains/nearby` - 获取附近毒株
- `POST /api/strain/publish` - 发布毒株
- `POST /api/strain/infect` - 感染毒株
- `GET /api/user/personal-stats` - 获取个人统计

## 📱 使用说明

### 主要功能
1. **发布毒株**: 选择类型，输入内容，获取位置，发布传播
2. **查看附近**: 发现身边的毒株，按距离和类型筛选
3. **数据统计**: 查看全局和个人传播数据
4. **个人中心**: 管理个人信息和设置

### 操作流程
1. 首次使用需要授权位置和用户信息
2. 在首页查看实时数据和附近毒株
3. 点击发布按钮创建新的毒株
4. 在附近页面发现和感染其他毒株
5. 在数据页面查看传播效果
6. 在个人中心管理账户和设置
