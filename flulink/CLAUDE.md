# FluLink 星尘共鸣版 - Claude Code 配置文件

## 项目概述
FluLink是一个基于"星空图谱"视觉隐喻的异步社交应用，每个用户都是孤独的星球，通过"星种"产生共鸣，在7天周期内汇聚成49人的"星团"。

## 核心设计理念
- **底层规则**: 遵循《道德经》吕尚版哲学思想
- **顶层表达**: 通过"星空图谱"视觉隐喻呈现
- **核心概念**: 共鸣、星团、星种、光度、光谱

## 技术架构
### 后端
- **语言**: Node.js + Express
- **数据库**: MongoDB (用户、星种、星团数据)
- **缓存**: Redis (星团计算结果、用户活跃状态)
- **认证**: JWT (多端适配支持)

### 前端
- **框架**: React + TypeScript
- **可视化**: D3.js (星空图谱渲染)
- **状态管理**: React Context + useReducer
- **UI组件**: Ant Design + 自定义星空组件

### 移动端
- **微信小程序**: 基于现有结构升级
- **iOS应用**: SwiftUI实现
- **Android应用**: React Native跨平台

## 核心算法
### 共鸣值计算
- 标签匹配(60%) + 互动历史(40%) + 随机因子(5%)
- 动态权重：初期标签权重高，成熟期互动权重高

### 星团生成
- 49人智能匹配 + 活跃度平衡 + 标签多样性约束
- 7天周期自动解散

### 星种演化
- 光度提升：评论+5，转发+3，点赞+1
- 光谱演化：根据用户标签调整权重
- 跃迁条件：光度≥70 + 光谱匹配≥0.6
